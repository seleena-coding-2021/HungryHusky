# `pages` Folder

This folder contains all the pages for each persona that will be displayed on the front end of the application. The pages starting with 00, 10, 20, and 30 are the home page for each user persona, and they include buttons that allow this persona to perform some sort of functionality. Once the user clicks on these buttons, it directs them to a different appropriate page that will allow the user to get certain information, post information, edit existing information in the database, or delete information from the database.
